# Era Shop — с нуля (для новичков)

Пошагово: что установить, где открыть проект и как увидеть веб в браузере.

---

## Шаг 1. Установить Flutter (один раз)

1. Скачай Flutter для Windows:  
   https://docs.flutter.dev/get-started/install/windows  
   (кнопка **Download Flutter SDK** — архив .zip)

2. Распакуй архив в папку **без пробелов**, например:  
   `C:\flutter`

3. Добавь Flutter в PATH:
   - Нажми **Win + R**, введи `sysdm.cpl`, Enter.
   - Вкладка **Дополнительно** → **Переменные среды**.
   - В «Переменные среды пользователя» выбери **Path** → **Изменить** → **Создать**.
   - Укажи путь к папке `bin` внутри Flutter, например:  
     `C:\flutter\bin`
   - **ОК** везде и закрой все окна.

4. Открой **новое** окно CMD или PowerShell и проверь:
   ```bash
   flutter doctor
   ```
   Должно быть без красных ошибок (жёлтые предупреждения — пока нормально).

---

## Шаг 2. Открыть проект

1. Открой папку с проектом в проводнике:  
   `C:\Users\buran\era_shop_clean`

2. В этой папке должен быть файл **run_web.bat** и папки **lib**, **web**, **assets**.

---

## Шаг 3. Запустить веб-версию

**Вариант А — двойной клик**

- Дважды нажми по файлу **run_web.bat**.  
- Откроется чёрное окно, пойдёт загрузка (первый раз может занять 1–3 минуты).  
- Когда в конце появится что-то вроде «Serving DevTools» и адрес — не закрывай окно.

**Вариант Б — из терминала**

1. В проводнике открой папку `C:\Users\buran\era_shop_clean`.
2. В адресной строке проводника введи `cmd` и нажми Enter — откроется терминал в этой папке.
3. Введи по очереди:
   ```bash
   flutter pub get
   flutter run -d web-server --web-port=8080
   ```
4. Дождись сообщения, что приложение запущено (не закрывай окно).

---

## Шаг 4. Открыть в браузере

1. Открой браузер (Chrome, Edge или другой).
2. В адресной строке введи:  
   **http://localhost:8080**  
   и нажми Enter.

Должен открыться интерфейс Era Shop (главная, меню внизу и т.д.).  
Если часть экранов пишет «ошибка» или «нет данных» — это из‑за того, что бэкенд ещё не настроен; интерфейс ты уже видишь.

---

## Если что-то не получилось

- **«flutter не является внутренней или внешней командой»**  
  → Flutter не в PATH или окно терминала было открыто до добавления PATH. Закрой все CMD/PowerShell, заново добавь в PATH путь к `C:\flutter\bin` (или куда распаковал) и открой терминал снова.

- **Окно run_web.bat сразу закрылось**  
  → Запусти из CMD (Шаг 3, вариант Б) — так ты увидишь текст ошибки и сможешь его скопировать.

- **В браузере пусто или «не удаётся подключиться»**  
  → Убедись, что чёрное окно с `flutter run` не закрыто и в нём нет красных ошибок. Адрес должен быть именно **http://localhost:8080**.
